# Bounce Infinite Remastered

A modern, desktop-ready recreation of the classic "Bounce Tales" game. Built with Electron and JavaScript, this project brings nostalgic gameplay to modern operating systems.

## Features
* **Cross-Platform:** Runs on Windows, macOS, and Linux.
* **Classic Mechanics:** Faithful recreation of physics and level design.
* **Enhanced Visuals:** Optimized for modern displays.

## How to Run
1. Clone the repo.
2. Run `npm install`.
3. Run `npm start`.
